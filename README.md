# tm1_bench_py
This opensource repository for the TM1 community. The solution was created to provide a Pyhonic TM1 benchmark model generator for automated testing, and performance testing internal tools.

## Requirements
* Python
* TM1py
* mdxpy
* Pandas

## Test Bench Introduction
To execute reproducible testing we added a small test bench solution.The solution is consist the following parts:

* scripts to build, execute, generate data and clean up at the end
* a TM1 model which could be elastically parameterized by size and data volume
* configuration.yaml to easily configrate for your needs
